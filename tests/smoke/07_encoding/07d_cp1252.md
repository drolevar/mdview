# mdview Smoke Test 07d - CP1252 (ANSI fallback)

This file is encoded in CP1252 (Windows-1252) with no BOM. Strict
UTF-8 decoding should fail on the high bytes; the encoding module
must fall through to CP_ACP and decode correctly on a Western Windows
system.

Western European accents:
- French: café, élève, hôtel, naïve
- German: Tür, Größe, Ärger
- Spanish: año, niño, España

Special CP1252 high bytes:
- Currency: € £ ¥
- Symbols: © ® ™
- Fractions: ¼ ½ ¾
- Math: × ÷ ± °

If you see these correctly, the CP_ACP fallback works.
If you see "?" or replacement characters, the decoder pipeline is
broken on this codepage.
